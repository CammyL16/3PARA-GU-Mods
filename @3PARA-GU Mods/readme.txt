Version 1.2: 3PARA-GU Mods
(19 Nov 2020)

Installation
------------
1. Unzip the folder and place in the root of Arma 3 installation. 
2. Add the mod name to the start-up using either a launcher or by adding "-mod=@3PARA-GU Mods". 
3. Copy the ".bikey" file to the "Keys" folder in Arma 3 root.
